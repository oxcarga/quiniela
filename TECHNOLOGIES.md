# Technologies Used

## Frontend (Main App)
- **Next.js** – React metaframework
- **React** – UI library
- **TypeScript** – Type-safe JavaScript
- **Tailwind CSS** – Utility-first CSS framework
- **PostCSS** – CSS transformations

## UI & Component Libraries
- **shadcn** – Component library built on Radix UI
- **Base UI** – Headless React components
- **Lucide React** – Icon library

## State Management & Data Fetching
- **Zustand** – Lightweight state management
- **TanStack React Query** – Server state management

## Backend/Firebase
- **Firebase** – Client SDK (Auth, Firestore)
- **Firebase Admin** – Server SDK for Functions
- **Firebase Functions** – Serverless backend
- **Firebase Emulators** – Local development (Auth, Firestore, Functions)

## Validation & Forms
- **Zod** – TypeScript-first schema validation

## Email
- **Resend** – Email service

## Development Tools
- **ESLint** – Code linting
- **Node.js** – JavaScript runtime

## Infrastructure
- **Firestore** – NoSQL database
- **Firestore Rules** – Security rules (custom file)
- **Firestore Indexes** – Database indexes config

## Fonts
- **Google Fonts** – Geist Sans/Mono typefaces

---

The project is a full-stack Next.js application with Firebase backend, Tailwind CSS styling, and a component-driven architecture using modern React patterns.

# Tecnologías Utilizadas

## Frontend (Aplicación Principal)
- **Next.js y React** – Framework 
- **TypeScript** – JavaScript seguro en tipos
- **Tailwind CSS** – Framework CSS orientado a utilidades
- **PostCSS** – Transformaciones de CSS

## Librerías de UI y Componentes
- **shadcn** – Librería de componentes construida sobre Radix UI
- **Base UI** – Componentes React sin estilos
- **Lucide React** – Librería de iconos
- **class-variance-authority** – Gestión de variantes de componentes
- **clsx** & **tailwind-merge** – Utilidades de clases CSS
- **tw-animate-css** – Utilidades de animación de Tailwind

## Gestión de Estado y Obtención de Datos
- **Zustand** – Gestión de estado ligera
- **TanStack React Query** – Gestión de estado del servidor

## Backend/Firebase
- **Firebase** – SDK de cliente (Auth, Firestore)
- **Firebase Admin** – SDK de servidor para Functions
- **Firebase Functions** – Backend sin servidor
- **Firebase Emulators** – Desarrollo local (Auth, Firestore, Functions)

## Validación y Formularios
- **Zod** – Validación de esquemas con TypeScript

## Email
- **Resend** – Servicio de correo electrónico

## Herramientas de Desarrollo
- **ESLint** – Linting de código
- **Node.js** – Runtime de JavaScript

## Infraestructura
- **Firestore** – Base de datos NoSQL
- **Firestore Rules** – Reglas de seguridad (archivo personalizado)
- **Firestore Indexes** – Configuración de índices de base de datos

## Fuentes
- **Google Fonts** – Tipografías Geist Sans/Mono

---

El proyecto es una aplicación full-stack de Next.js con backend de Firebase, estilos de Tailwind CSS y una arquitectura basada en componentes usando patrones modernos de React.
